Title: Entwicklung einer Webanwendung für online Produkt presentation, verkauf und verwaltung.

## Analyse ##: 
Benötigt wird eine multifunktionale Website die unabhängig von andere Platformen ist und        
kann jeder E-Commerce Model umsetzen. Administration Dashboard die ermöglicht die Website 
inhalt und layout zu verwalten. Ein format agnostic schnittstelle für JSON, CSS, XML 
importierung und speicherung in Datenbanken.


## IST- und SOLL-Zustand ##
- [Ist-Zustand]: Aktuell besteht keine möglichkeit daten aus formate wie JSON, CSS oder XML direkt über
                 admindashboard zu importieren, was verursacht ineffizienter manueler und fehlerhafte daten erfassung.
- [Soll-Zustand]: Nach der Projekt umsetzung soll eine schnittstelle bereitgestellt werden, die ermöglicht 
                  schnelle und fehlerfreie daten importierung und automatische speicherung in Datenbanken.
